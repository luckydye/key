# key

```
Command Line Interface to a local or remote keepass database.

Optional Environment Variables:
  KEEPASSDB - url to the keepass database (examples: file:///bucket/path or s3://s3.example.com/bucket/path)
  KEEPASSDB_KEYFILE - path to the keyfile for the keepass database
  KEEPASSDB_PASSWORD - password for the keepass database

Usage:
  key [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  get         Get an entry from the database
  help        Help about any command
  list        List all entries in the database

Flags:
  -h, --help   help for key

Use "key [command] --help" for more information about a command.
```
